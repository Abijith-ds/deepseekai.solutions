# Cloudflare Pages Deployment Guide

Follow these steps to deploy DeepSeek AI Solutions to Cloudflare Pages:

## Deployment Steps

1. Go to **[dash.cloudflare.com/workers-and-pages/create](https://dash.cloudflare.com/workers-and-pages/create)**
2. Select **Pages → Connect to Git**
3. Authorize GitHub and select `Abijith-ds/deepseekai.solutions`
4. Set build branch to `cloudflare-deployment`
5. Leave build command empty (static site)
6. Set publish directory to `.` (root)
7. Deploy!

## Post-Deployment

Your site will be live at a Cloudflare Pages URL instantly. You can then connect your custom domain `deepseekai.solutions` in Cloudflare settings.
